# Antigravity CLI

**Читати іншими мовами:** [English](./README.md) · [Українська](./README-UA.md)

Antigravity CLI розуміє вашу кодову базу, вносить зміни з вашого дозволу та виконує команди — безпосередньо з вашого терміналу.

- **Офіційна документація**: [antigravity.google/docs/cli-overview](https://antigravity.google/docs/cli-overview)
- **Офіційний сайт**: [antigravity.google/product/antigravity-cli](https://antigravity.google/product/antigravity-cli)

![Antigravity CLI Demo](agy-cli-demo.gif)

---

Antigravity CLI переносить основні можливості системи Antigravity 2.0 (багатокрокові міркування, редагування декількох файлів, виклик інструментів та збереження історії сесій) безпосередньо у ваш термінал. Продукт оптимізовано для роботи за допомогою клавіатури та віддалених SSH-сесій з мінімальним споживанням системних ресурсів.

## Features at a Glance

| Функція | Antigravity CLI | Antigravity 2.0 |
| :--- | :--- | :--- |
| **Основний фокус** | Швидкість, ефективність клавіатури, низькі накладні витрати | Повнота можливостей, візуальна оркестрація, керування проєктами |

| **Агентний рушій** | Спільне ядро агентного рушія | Спільне ядро агентного рушія |

## Інтеграція

- **Спільне ядро**: Обидва інтерфейси працюють на одному спільному агентному рушії. Покращення автоматично застосовуються для обох версій.
- **Спільні налаштування**: Налаштування та дозволи синхронізуються в обох напрямках.
- **Експорт сесій**: Можливість експортувати термінальні сесії в графічний інтерфейс Antigravity 2.0 для продовження роботи.

## Встановлення

### macOS / Linux
```bash
curl -fsSL https://antigravity.google/cli/install.sh | bash
```

### Windows PowerShell
```powershell
irm https://antigravity.google/cli/install.ps1 | iex
```

### Windows CMD
```cmd
curl -fsSL https://antigravity.google/cli/install.cmd -o install.cmd && install.cmd && del install.cmd
```

## Автентифікація

CLI проходить автентифікацію через системну зв'язку ключів, автоматично повертаючись до входу через Google (Google Sign-In) за відсутності активної сесії.

- **Локально**: Автоматично відкриває ваш браузер за замовчуванням.
- **Віддалено / через SSH**: Виявляє SSH-сесії та виводить URL-адресу для авторизації, щоб завершити вхід локально.
- **Вихід**: Виконайте команду `/logout` для очищення збережених облікових даних.

> [!NOTE]
> Для корпоративного доступу підключіть свій проєкт GCP під час першого запуску. Детальніше див. на сторінці Enterprise.
